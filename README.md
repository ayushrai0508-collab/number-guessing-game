# Number Guessing Game 🎯

This is a simple Python mini-project where the user has to guess the correct number.

## How it works

* The program stores a fixed number.
* The user enters a guess.
* If the guess is lower than the number, it shows **"Too low"**.
* If the guess is higher than the number, it shows **"Too high"**.
* The game continues until the user guesses correctly.

## Concepts Used

* Variables
* User Input
* While Loop
* If-Else Conditions

## Technologies Used

* Python 3

## Example Output

Enter the number: 1
Too low
Enter the number again: 3
Too high
Enter the number again: 2
You win 🌟
